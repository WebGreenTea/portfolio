E-meeting application

Technology Stack -  Laravel Lumen, MySQL, Node.js, Onesignal

This is an application for web and mobile to management meetings. This application has many features for the convenience of meeting management. I have developed the backend of this project, including database design, API integration, and more.

The main framework used to develop the API for this project is Lumen (PHP Micro-Framework By Laravel). and choose to use Relational Database as MySQL.

Use the Onesignal service to send notifications to mobile application users.

There is also an email sending feature that includes instant and scheduled emails. Instant email uses Lumen, but scheduled email uses Node.js using node-cron to deliver emails that are on time. send Use Handlebars to create the look of an email and send it using Nodemailer.
